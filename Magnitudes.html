Magnitudes

Una magnitud es cualquier propiedad medible de un sistema físico. Esta puede ser representada por un valor numérico seguido de una unidad asociada. Por ejemplo, para el caso de una longitud \(L\):

\( L = 2.50 m \)

En esta expresión podemos apreciar el valor numérico \(2.50\) más la unidad \(m\) metros. Este ejemplo corresponde a una magnitud escalar. Todas las magnitudes escalares se pueden representar de esta forma, pero también existe otro tipo de magnitudes para las cuales se necesita brindar un poco más de información, de forma de poder representar todas sus características. Son las llamadas magnitudes vectoriales.

Un ejemplo concreto de magnitud vectorial es la velocidad. La velocidad de un objeto en movimiento se encuentra determinada por su rapidez, dirección y sentido. Estas caracteristicas mencionadas definen a un vector. Si observamos el movimiento un movil a lo largo de diferentes trayectorias, se puede ver que el vector velocidad puede tener una rapidez constante pero ir cambiando de dirección como se observa en la animación 1
