Magnitudes

Una magnitud es cualquier propiedad medible de un sistema físico. En este sentido puede ser representada por un valor numérico seguido de una unidad asociada. Por ejemplo, para el caso de una longitud \(L\):

\( L = 2.50 m \)

En esta expresión podemos apreciar el valor numérico \(2.50\) más la unidad \(m\) metros. Este ejemplo corresponde a una magnitud escalar. Todas las magnitudes escalares se pueden representar de esta forma, pero existe otro tipo de magnitudes para las cuales se necesita brindar un poco más de información, de forma de poder representar todas sus características. estas son las llamadas magnitudes vectoriales.
